# 🔢 Operators in Java

This folder contains Java programs based on different types of operators.

---

## 📌 Topics Covered

* ➕ Arithmetic Operators (`+`, `-`, `*`, `/`, `%`)
* 🔍 Relational Operators (`>`, `<`, `==`, `!=`)
* 🔗 Logical Operators (`&&`, `||`, `!`)
* 📝 Assignment Operators (`=`, `+=`, `-=`, etc.)

---

## 💡 Example Concepts

* Performing calculations using arithmetic operators
* Comparing values using relational operators
* Combining conditions using logical operators

---

## 🚀 Goal

To understand how operators work in Java and how they are used in real programs.
