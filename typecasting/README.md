# 🔄 Typecasting in Java

This folder contains Java programs related to type conversion.

---

## 📌 Topics Covered

* 🔼 Implicit Typecasting (Widening)

  * Automatic conversion (e.g., `int` → `double`)
* 🔽 Explicit Typecasting (Narrowing)

  * Manual conversion (e.g., `double` → `int`)

---

## 💡 Example Concepts

* Converting smaller data types to larger types
* Handling data loss in narrowing conversions
* Using casting operators in Java

---

## 🚀 Goal

To understand how data types can be converted and how typecasting works internally in Java.
