# 🧩 Pattern Printing in Java

This repository contains various **pattern printing programs in Java**, designed to strengthen logic building and mastery of loops.

---

## 🚀 Topics Covered

* Nested loops
* Pattern logic (rows & columns)
* Star patterns
* Hollow patterns
* Pyramid structures

---

## 📂 Patterns Included

### 🔹 Basic Patterns

* `pattern1.java` – Simple star pattern
* `pattern4.java` – Advanced pattern variation

---

### 🔹 Solid Patterns

* `solidRectangle.java` – Solid rectangle pattern
* `solidRightAngleTriangle.java` – Right-angled triangle
* `solidPyramid.java` – Pyramid structure

---

### 🔹 Hollow Patterns

* `hollowRectangle.java` – Hollow rectangle
* `hollowRightAngleTriangle.java` – Hollow triangle

---

### 🔹 Inverted Patterns

* `invertedPyramid.java` – Inverted pyramid
* `invertedRightAngleTriangle.java` – Inverted triangle

---

## 💡 Key Learning

* Breaking patterns into:

  * Spaces
  * Symbols (`*`)
* Understanding how rows and columns interact
* Building logic step by step


---

## 🎯 Goal

To build a strong foundation in **problem-solving and pattern recognition**, essential for DSA and coding interviews.
