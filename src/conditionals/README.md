🔀 Conditionals in Java

* This file demonstrates the use of conditional statements in Java, which help control the flow of a program based on different conditions.

📌 Topics Covered
* if statement
* if-else statement
* if-else if ladder
* Nested if
* Ternary Operator (?:)
* switch statement

🧠 Why Conditionals?

Conditionals allow a program to make decisions. Based on certain conditions, different blocks of code are executed.

⚠️ Important Notes
* Always use break in switch to avoid fall-through.
* Ternary operator is best for simple conditions.
* For complex logic, prefer if-else for readability.

🚀 Conclusion
* Conditionals are a core concept in programming and are essential for decision-making in real-world applications. Mastering them will help you write efficient and logical code.
